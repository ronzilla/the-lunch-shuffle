Welcome to the The Lunch Shuffle!

This tool will shuffle a list of people into groups. The shuffler will try to match up people with little in common who haven't been matched up before.

  1. Upload a file

    • This file should be text, tab-delimited.
    • First row is the header.
    • First 3 columns are First, Last, and Email.
    • Columns beyond 3 are optional parameters for assigning mismatch points. The mismatch value goes in parentheses after the parameter name.
    • A sample data file might provide some clues. In the sample data, people from the same Department get 5 mismatch points, people from the same Group get 10, and people with the same Manager get 20. If two people score high mismatch values versus each other, they'll tend to not be matched up.
